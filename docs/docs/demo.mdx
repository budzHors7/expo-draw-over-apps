# Demo

The repo includes a working Expo example app in `example/`.

Run the example after installing dependencies and building the native app for Android:

```bash
cd example
npm install
npm run android
```

The app lets you:

- check overlay permission
- open the Android overlay settings screen
- launch each registered bubble fixture
- toggle edge hide for newly shown bubbles
- close the active bubble and release its overlay surface
- close visible bubbles when each fixture is done
- change example counters from the app or from the bubble
- keep counter and resize values in native shared state
- refresh shared values when the app returns to the foreground
- open the app from a bubble
- test the long-press remove menu

The fixture cards come from `BUBBLE_PLAYGROUND_TEMPLATES`. When you add a new fixture, add it there first and then update the fixture docs.
